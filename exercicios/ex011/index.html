Usando links Externos

Você pode acessar o meu repositório público no GitHub

Para adicionar um endereço de link ao clicar na palavra se usa a tag a e colocando o link no href
(dica: para acessar o link sem sair da pagina original se usa a tag target="_blank")
< a href="link" target="_blank">
é legal colocar tambem rel="external", para indicar para o navegador que é um, link externo
< a href="link" target="_blank" rel="external"

Usando links Internos

Esta é a primeira página do site. Se você quiser pode acessar também a minha segunda pagina

Para entrar em outro site, basta criar outro arquivo html (pag002.html)
usar a tag a href="pag002" e é indicado colocar rel="next"
para o navegador entender que o link é próxima página e na outra página colocar rel="prev" para o navegador entender que o link leva para a pagina anterior

Você também pode acessar a nossa página de notícias, clique aqui

Coloca target="_self" para indicar que é um link para acessar na mesma página
